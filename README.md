# 🎓 Student Result Management System

## 📌 Description
This is a web-based Student Result Management System developed using Django (Python). It helps manage student records, marks, and results efficiently.

## 🚀 Features
- Add new students
- Update student marks
- View student results
- Simple and user-friendly interface

## 🛠️ Technologies Used
- Python
- Django
- HTML, CSS
- SQLite

## 📂 Project Structure
- Resultapp → Main application
- StudentResultManagementss → Project settings
- images → Static images
